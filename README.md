# JARVIS-Voice-Assistant-System



## requirements

- python 3.11 or higher 


## How to run ?



1. create a virtual 
environment:
    ```
     python create -m venv jarvis python=3.11 -y
    ```

2. Activate virtual environment
    ```
    conda activate jarvis
    ```

3. Install required pakages

    ```
    pip install -r requirements.txt
    ```
4. 

    ```
    python jarvis.py
    ```
